# v3.1.0 Release Notes — Web UI License Activation

**Released: March 18, 2026**

## What's New

### License Key Paste-to-Activate
- **Summary**: License keys can now be pasted directly into the license gate UI
- **Details**: Added a textarea input and "Activate License" button to the license gate overlay. Users can paste their license key directly into the web interface without needing SSH access or running install scripts.
- **Impact**: Dramatically simplifies license installation — no terminal access required

### One-Click Machine ID Copy
- **Summary**: Machine ID can be copied with a single click from the license gate
- **Details**: Click-to-copy with clipboard API and fallback. Shows "Copied!" confirmation feedback.
- **Impact**: Faster license provisioning workflow

## Technical Changes
- Added `/api/activate_license` POST endpoint to Express server
- Added `fs` module import for license key persistence
- License key set in `process.env.LICENSE_KEY` and persisted to `/app/data/license_key`
- Frontend `activateLicense()` function with validation, error handling, and status feedback
- CSS styles for license input section, activate button, divider, and status messages
