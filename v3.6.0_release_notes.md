# v3.6.0 Release Notes — Self-Test Endpoints

**Released: April 29, 2026**

## What's New

### `/api/selftest` and `/api/selftest/checklist`
- **Summary**: New runtime self-test endpoints mirror the contract used by Chart Toppers, so Docforge can produce a uniform Installation Test Report PDF for Win It In A Minute.
- **Automated checks** (8 total):
  - **Infrastructure**: License valid, OSC listener configured, QLab OSC target configured
  - **License Persistence**: Volume copy present, Host backup present, Tiers in sync
  - **Stream Deck Configuration**: Red team cue configured, Blue team cue configured
- **Manual checklist** (15 items): Setup, Round 1 SD OSC, Scoring, OSC, Sync, Recovery
- **Impact**: WIIAM is now a first-class project in Docforge's test report flow — pick "Win It In A Minute" in the project dropdown to generate a branded PDF.

## Technical Changes
- `server.js`: new `/api/selftest` and `/api/selftest/checklist` endpoints. Version cached at startup as `APP_VERSION` to avoid repeated reads of read-only `package.json` inside the container.
