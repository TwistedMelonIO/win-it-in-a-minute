# v3.6.1 Release Notes — Bulk Docker Updater Script

**Released: April 29, 2026**

## What's New

### `update-twisted-melon.sh` — one-command refresh for any demo machine
- **Summary**: New top-level helper script that installs OR updates Chart Toppers, Win It In A Minute, and DocForge in a single run. Idempotent: clones missing repos, pulls existing ones, then rebuilds containers.
- **Usage on any venue / demo machine**:
  ```bash
  bash <(curl -fsSL https://raw.githubusercontent.com/TwistedMelonIO/win-it-in-a-minute/main/update-twisted-melon.sh)
  ```
- **What it does**:
  - Verifies Docker daemon is running
  - For each project: clones to `~/<project>` if missing, otherwise `git pull --ff-only`
  - Ensures `~/.twisted-melon/<project>/` host backup dirs exist (license persistence)
  - `docker compose up -d --build` per project
  - Reports the resulting version of each container
- **License keys are NOT touched** — the dual-tier persistence (volume + host backup) restores them automatically on container recreate.

## Why this lives in WIIAM
- The script is generic and updates all three projects, but it has to live somewhere with a stable raw GitHub URL. WIIAM was the most recently active repo so it gets the home.
