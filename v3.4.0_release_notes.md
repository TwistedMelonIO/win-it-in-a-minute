# v3.4.0 Release Notes — Always-Export License Certificate

**Released: April 25, 2026**

## What's New

### Always-export `.docforge` certificate file
- **Summary**: The license installer now writes a `.docforge` certificate file to the Desktop after a successful activation.
- **Details**: Added a docforge export block to `install_license.sh`. After a successful activation, the script prompts for the licensee name and expiry days (both optional — Enter to skip), then unconditionally writes `~/Desktop/win-it-in-a-minute-license.docforge` containing the machine ID, license key, licensee, and expiry metadata. Brings WIIAM in line with chart-toppers and code-breakers.
- **Impact**: Operators end up with a portable certificate file they can hand off to Docforge for cert generation, every time.

## Technical Changes
- `install_license.sh`: new docforge export block in the success branch; mirrors chart-toppers' schema with `project_id: 'win-it-in-a-minute'`.
