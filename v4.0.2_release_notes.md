# Win It In A Minute v4.0.2

## Fully offline-safe fonts

Same offline hardening applied to Chart Toppers v8.0.4 — Google Fonts (Teko 500/600/700, Russo One, Barlow 400/500/600/700) are now bundled with the image instead of fetched from `fonts.googleapis.com`.

WIIAM never had the render-blocking script issue Chart Toppers had (it already used the `media="print" onload` non-blocking pattern), so the main UI didn't hang offline — it just rendered with fallback system fonts. Now it renders correctly even on machines with no internet.

## Files changed
- `public/vendor/fonts.css` + `public/vendor/fonts/*.woff2` — bundled latin-subset fonts
- `public/index.html`, `public/settings.html` — replaced Google Fonts links with local `vendor/fonts.css`
- `package.json`, `public/index.html` (×2: gate + version span), `public/settings.html` — version bump to 4.0.2
