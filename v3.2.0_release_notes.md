# v3.2.0 Release Notes — Bitfocus Companion OSC Integration

**Released: March 20, 2026**

## What's New

### Expanded OSC Command Set for Bitfocus Companion
- **Summary**: Full per-team score control via OSC commands
- **Details**: Added individual OSC commands for incrementing and decrementing each team's score, plus per-team reset commands. The original `/wiiam/reset` command still works for full game reset. All available OSC commands are now printed at server startup for easy reference.
- **Impact**: Operators can now control scores from Bitfocus Companion buttons without using the web UI

### New OSC Commands
| Command | Action |
|---------|--------|
| `/wiiam/red/up` | Increment Red team score |
| `/wiiam/red/down` | Decrement Red team score |
| `/wiiam/blue/up` | Increment Blue team score |
| `/wiiam/blue/down` | Decrement Blue team score |
| `/wiiam/reset/red` | Reset Red team only |
| `/wiiam/reset/blue` | Reset Blue team only |
| `/wiiam/reset` | Full game reset (existing) |

## Bug Fixes
- Settings page footer now shows license status immediately on page load (before passcode entry)

## Technical Changes
- OSC handler refactored from single if-statement to switch-based dispatch
- All OSC messages now logged via `addLog()` instead of `console.log()`
- Unknown OSC addresses logged as warnings
- Server startup prints formatted block of all available OSC commands
