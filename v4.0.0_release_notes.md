# v4.0.0 Release Notes — Win It In A Minute 4.0 milestone

## Summary

Win It In A Minute crosses to its 4.0 milestone. Stable feature set for live show use, with QLab + Companion integration, license-gated activation, persistent settings, self-test endpoints for installation verification, and an improved selftest payload that surfaces machine identity.

## Changes in this release

- **`/api/selftest`** now reports `machineId` and `licensee` alongside the existing version / hostname / project metadata, so installation reports can confirm the licensed machine and licensee in one call.
- Version bump across all surfaces: `package.json`, license gate, header version pill, settings page.

## Carried forward from the 3.x line

- Self-test endpoints + unified auto-test entry point (3.5.x / 3.6.0)
- Bulk Docker updater script (3.6.1)
- License persistence via named Docker volume + host-side backup
- QLab + Bitfocus Companion OSC integration

## Upgrade

```bash
docker compose up -d --build
```

License key, machine ID, and persistent settings survive the rebuild via the named Docker volume.
