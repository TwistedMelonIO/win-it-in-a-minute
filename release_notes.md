Win It In A Minute v2.1.0

Enhanced version with improved stability and user experience:

## Features
- Split-screen scoring UI with Red team on left, Blue team on right  
- Large score displays for easy reading from distance
- Real-time sync across multiple browser windows via WebSocket
- OSC integration with QLab for automatic text cue updates
- Keyboard shortcuts for quick scoring (Q/A for Red, P/L for Blue)
- Docker support for easy deployment
- Settings panel for QLab cue number configuration

## Improvements in v2.1.0
- Enhanced WebSocket connection stability
- Improved OSC message handling and error recovery
- Better responsive design for various screen sizes
- Refined neon arena visual aesthetics
- More robust score synchronization across clients
- Enhanced Docker configuration for production deployment

This version builds on the v2.0 foundation with significant reliability
and user experience improvements for professional theater environments.
